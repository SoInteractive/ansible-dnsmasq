Example usage
-------------

Use it in a playbook as follows:
```yaml
- hosts: all
  become: true
  roles:
    - SoInteractive.dnsmasq
```

Have a look at the [defaults/main.yml](defaults/main.yml) for role variables
that can be overridden.
